📋 JavaScript 常用事件监听速查表

涵盖鼠标、键盘、表单、窗口、触摸、拖拽、剪贴板及媒体事件

🖱 鼠标 ⌨ 键盘 📝 表单 🪟 窗口 👆 触摸 ↕ 拖拽 📋 剪贴板 🎬 媒体
事件名称 分类 触发时机 适用元素 冒泡 代码示例
click 鼠标 鼠标单击（按下并释放） 几乎所有元素 ✔ 是 element.addEventListener('click', fn)
dblclick 鼠标 鼠标双击 几乎所有元素 ✔ 是 element.addEventListener('dblclick', fn)
mousedown 鼠标 鼠标按键按下（未释放） 几乎所有元素 ✔ 是 element.addEventListener('mousedown', fn)
mouseup 鼠标 鼠标按键释放 几乎所有元素 ✔ 是 element.addEventListener('mouseup', fn)
mousemove 鼠标 鼠标在元素内移动（高频触发） 几乎所有元素 ✔ 是 element.addEventListener('mousemove', fn)
mouseenter 鼠标 鼠标进入元素（不冒泡） 几乎所有元素 ✘ 否 element.addEventListener('mouseenter', fn)
mouseleave 鼠标 鼠标离开元素（不冒泡） 几乎所有元素 ✘ 否 element.addEventListener('mouseleave', fn)
mouseover 鼠标 鼠标进入元素（会冒泡） 几乎所有元素 ✔ 是 element.addEventListener('mouseover', fn)
mouseout 鼠标 鼠标离开元素（会冒泡） 几乎所有元素 ✔ 是 element.addEventListener('mouseout', fn)
contextmenu 鼠标 右键菜单触发 几乎所有元素 ✔ 是 element.addEventListener('contextmenu', fn)
wheel 鼠标 鼠标滚轮滚动 几乎所有元素 ✔ 是 element.addEventListener('wheel', fn)
keydown 键盘 键盘按键按下（按住会重复触发） 可聚焦元素、document ✔ 是 document.addEventListener('keydown', fn)
keyup 键盘 键盘按键释放 可聚焦元素、document ✔ 是 document.addEventListener('keyup', fn)
keypress 键盘 字符键按下（已废弃，不推荐） 可聚焦元素、document ✔ 是 document.addEventListener('keypress', fn)
submit 表单 表单提交时触发 <form> ✔ 是 form.addEventListener('submit', fn)
reset 表单 表单重置时触发 <form> ✔ 是 form.addEventListener('reset', fn)
input 表单 输入框值发生变化（实时触发） <input>、<textarea>、<select> ✔ 是 input.addEventListener('input', fn)
change 表单 值改变且失去焦点时触发 <input>、<textarea>、<select> ✔ 是 input.addEventListener('change', fn)
focus 表单 元素获得焦点（不冒泡） 可聚焦元素 ✘ 否 input.addEventListener('focus', fn)
blur 表单 元素失去焦点（不冒泡） 可聚焦元素 ✘ 否 input.addEventListener('blur', fn)
focusin 表单 元素获得焦点（会冒泡） 可聚焦元素 ✔ 是 input.addEventListener('focusin', fn)
focusout 表单 元素失去焦点（会冒泡） 可聚焦元素 ✔ 是 input.addEventListener('focusout', fn)
DOMContentLoaded 窗口 DOM 树构建完成（不等资源加载） document ✘ 否 document.addEventListener('DOMContentLoaded', fn)
load 窗口 页面所有资源加载完成 window、<img>、<iframe> ✘ 否 window.addEventListener('load', fn)
beforeunload 窗口 页面即将卸载（可提示用户） window ✘ 否 window.addEventListener('beforeunload', fn)
resize 窗口 窗口尺寸变化（高频触发） window ✘ 否 window.addEventListener('resize', fn)
scroll 窗口 元素或页面滚动（高频触发） window、可滚动元素 ✘ 否 window.addEventListener('scroll', fn)
hashchange 窗口 URL 的 hash 部分变化 window ✘ 否 window.addEventListener('hashchange', fn)
popstate 窗口 浏览器历史记录变化（前进/后退） window ✘ 否 window.addEventListener('popstate', fn)
error 窗口 资源加载失败或 JS 运行时错误 window、<img>、<script> ✘ 否 window.addEventListener('error', fn)
touchstart 触摸 手指触摸屏幕 几乎所有元素 ✔ 是 element.addEventListener('touchstart', fn)
touchmove 触摸 手指在屏幕上滑动（高频触发） 几乎所有元素 ✔ 是 element.addEventListener('touchmove', fn)
touchend 触摸 手指离开屏幕 几乎所有元素 ✔ 是 element.addEventListener('touchend', fn)
touchcancel 触摸 触摸被中断（如弹窗、切换应用） 几乎所有元素 ✔ 是 element.addEventListener('touchcancel', fn)
dragstart 拖拽 开始拖拽元素 draggable 元素 ✔ 是 element.addEventListener('dragstart', fn)
drag 拖拽 拖拽过程中持续触发 draggable 元素 ✔ 是 element.addEventListener('drag', fn)
dragend 拖拽 拖拽结束 draggable 元素 ✔ 是 element.addEventListener('dragend', fn)
dragover 拖拽 拖拽元素悬停在目标上（需 preventDefault） 放置目标元素 ✔ 是 dropZone.addEventListener('dragover', fn)
drop 拖拽 拖拽元素在目标上释放 放置目标元素 ✔ 是 dropZone.addEventListener('drop', fn)
copy 剪贴板 用户复制内容时触发 几乎所有元素 ✔ 是 document.addEventListener('copy', fn)
cut 剪贴板 用户剪切内容时触发 几乎所有元素 ✔ 是 document.addEventListener('cut', fn)
paste 剪贴板 用户粘贴内容时触发 几乎所有元素 ✔ 是 document.addEventListener('paste', fn)
play 媒体 媒体开始播放 <audio>、<video> ✘ 否 video.addEventListener('play', fn)
pause 媒体 媒体暂停 <audio>、<video> ✘ 否 video.addEventListener('pause', fn)
ended 媒体 媒体播放完毕 <audio>、<video> ✘ 否 video.addEventListener('ended', fn)
timeupdate 媒体 播放进度更新（高频触发） <audio>、<video> ✘ 否 video.addEventListener('timeupdate', fn)
loadeddata 媒体 媒体首帧加载完成 <audio>、<video> ✘ 否 video.addEventListener('loadeddata', fn)

💡 提示：使用 removeEventListener 可移除监听器； 对于高频事件（scroll / resize / mousemove / touchmove），建议使用 节流 (throttle) 或 防抖 (debounce) 优化性能。
