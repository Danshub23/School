📚 综合应用示例

以下示例假设已正确创建 Phaser 游戏场景，并在其中定义 this.player 为一个精灵（Sprite）且已启用物理系统（如需碰撞）。

示例1：让角色面向鼠标并前进10步

// 获取鼠标坐标
let mouseX = this.input.activePointer.x;
let mouseY = this.input.activePointer.y;

// 计算角色到鼠标的角度（弧度）
let angleRad = Phaser.Math.Angle.Between(this.player.x, this.player.y, mouseX, mouseY);

// 设置角色方向（Phaser 弧度制，也可用 .angle 角度制）
this.player.rotation = angleRad;

// 前进10步
let steps = 10;
this.player.x += Math.cos(angleRad) * steps;
this.player.y += Math.sin(angleRad) * steps;

示例2：控制角色跟随鼠标移动（每帧更新）

// 在 update() 循环中执行
let speed = 5;
let mouseX = this.input.activePointer.x;
let mouseY = this.input.activePointer.y;

let angle = Phaser.Math.Angle.Between(this.player.x, this.player.y, mouseX, mouseY);
this.player.x += Math.cos(angle) * speed;
this.player.y += Math.sin(angle) * speed;

示例3：鼠标按下时隐藏角色，松开时显示

if (this.input.activePointer.isDown) {
    this.player.setVisible(false);
} else {
    this.player.setVisible(true);
}

示例4：检测角色是否碰到鼠标指针（圆形近似），并改变大小

let mouseX = this.input.activePointer.x;
let mouseY = this.input.activePointer.y;
let distance = Phaser.Math.Distance.Between(this.player.x, this.player.y, mouseX, mouseY);

if (distance < this.player.width / 2) {
    this.player.setScale(1.5);   // 放大到150%
} else {
    this.player.setScale(1.0);   // 恢复原大小
}

示例5：获取角色到舞台中心的Scratch风格方向角

let centerX = this.cameras.main.width / 2;
let centerY = this.cameras.main.height / 2;
let angleRad = Phaser.Math.Angle.Between(this.player.x, this.player.y, centerX, centerY);
let angleDeg = Phaser.Math.RadToDeg(angleRad);
let scratchDirection = (90 - angleDeg + 360) % 360;
console.log(`角色指向舞台中心的方向(Scratch格式): ${scratchDirection}°`);

注：以上代码适用于 Phaser 3.90.0，使用前请确保已正确初始化游戏、加载资源并创建了 this.player 精灵。部分碰撞检测需要开启物理引擎。
