HTML <input> 所有类型一览

类型 代码示例 渲染效果
button <input type="button" value="按钮">
checkbox <input type="checkbox"> 选项 选项
color <input type="color">
date <input type="date">
datetime-local <input type="datetime-local">
email <input type="email" placeholder="name@example.com">
file <input type="file">
hidden <input type="hidden" value="隐藏数据"> （隐藏元素，不可见）
image <input type="image" src="图标.svg" alt="图片按钮" width="32">
month <input type="month">
number <input type="number" value="5">
password <input type="password" placeholder="输入密码">
radio <input type="radio" name="group"> 男 <input type="radio" name="group"> 女 男 女
range <input type="range" min="0" max="10">
reset <input type="reset" value="重置">
search <input type="search" placeholder="搜索...">
submit <input type="submit" value="提交">
tel <input type="tel" placeholder="123-456-7890">
text <input type="text" placeholder="普通文本">
time <input type="time">
url <input type="url" placeholder="https://example.com">
week <input type="week">

📌 说明：

  • hidden 类型在页面中不可见，此处用文字说明。
  • image 类型使用内联 SVG 作为示例，实际使用时请替换为有效图片路径。
  • 部分类型（如日期、颜色等）的显示效果取决于浏览器支持，不支持时会降级为普通文本输入框。
  • 复选框（checkbox）和单选按钮（radio）的 name 属性用于分组，此处仅展示基本外观。
  • reset 按钮需要位于 <form> 内并配合其他输入才能体现重置效果，此处仅展示按钮样式。
