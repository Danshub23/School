📊 HTML 表格知识点总结

① 基础标签与结构

表格核心标签
标签 描述 示例用法
<table>定义表格容器<table> ... </table>
<tr>定义行<tr><td>...</td></tr>
<th>表头单元格（加粗居中）<th>姓名</th>
<td>标准数据单元格<td>张三</td>
<caption>表格标题（显示在表格上方）<caption>销售记录</caption>
<thead>表头组（可包含多行）包裹标题行
<tbody>表格主体组包裹数据行
<tfoot>表尾组（如汇总）包裹统计行
💡 提示：<thead> / <tbody> / <tfoot> 仅用于语义分组，不影响实际渲染，但有助于样式和屏幕阅读器。

③ 合并单元格（colspan / rowspan）

🎓 课程表示例（合并演示）
时间 星期一 星期二 星期三
上午 数学 语文 英语
物理 化学 生物
下午 体育（两节连上） 音乐

  • colspan="2" 让“体育”单元格合并两列

④ 传统属性 vs 现代 CSS

属性对照表（推荐使用 CSS）
属性作用CSS 替代方案
border="1"设置边框border: 1px solid #999;
cellpadding="10"单元格内边距padding: 10px; (在 th/td 上)
cellspacing="0"单元格间距border-collapse: collapse; 或 border-spacing: 0;
align="center"表格水平对齐margin: 0 auto; (对表格本身) / text-align (对内容)
width="500"表格宽度width: 500px; 或百分比
⚠️ 大多数早期属性（如 bgcolor、align）在 HTML5 中已不推荐，应优先使用 CSS 控制样式。
