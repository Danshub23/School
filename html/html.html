📋 常用 HTML 标签速查表（按类别分组）

表格中每个类别上方都有醒目的分组标题，方便查阅。

标签 描述 类别
📄 文档声明
<!DOCTYPE>文档类型声明，必须放在 HTML 文档最前面文档声明
📁 文档基础
<html>HTML 文档的根元素文档基础
<body>文档内容容器，包含所有可见内容文档基础
🔧 文档元数据
<head>包含文档元数据的容器文档元数据
<title>定义文档标题，显示在浏览器标题栏或标签页文档元数据
<base>指定文档中所有相对 URL 的基础 URL文档元数据
<link>定义与外部资源的关系（如样式表、图标）文档元数据
<meta>定义文档的元数据（如字符集、描述、视口设置）文档元数据
<style>包含文档的样式信息（CSS）文档元数据
🗂️ 内容分区
<article>表示独立的、可复用的文章或内容块内容分区
<section>表示文档中的一个区域（通常带标题）内容分区
<nav>定义导航链接的容器内容分区
<aside>定义与主内容间接相关的附属内容（如侧边栏）内容分区
<h1>-<h6>六级标题，<h1> 最重要，<h6> 最不重要内容分区
<header>定义文档或节的页眉，通常包含标题或导航内容分区
<footer>定义文档或节的页脚，通常包含版权、联系信息内容分区
<main>定义文档的主要内容，每个页面仅可使用一次内容分区
<address>定义联系信息（如作者、所有者、联系方式）内容分区
📝 文本内容
<p>定义段落文本内容
<hr>定义主题分割线（水平线）文本内容
<pre>定义预格式化的文本（保留空格和换行）文本内容
<blockquote>定义长的引用块（通常有缩进）文本内容
<div>定义块级容器，用于布局或样式分组文本内容
<figure>定义独立的引用内容（如图表、插图、代码片段）文本内容
<figcaption>定义 <figure> 元素的标题或说明文本内容
📋 列表
<ul>定义无序列表（项目符号）列表
<ol>定义有序列表（编号）列表
<li>定义列表项，用于 <ul> 或 <ol>列表
<dl>定义描述列表列表
<dt>定义描述列表中的术语列表
<dd>定义描述列表中术语的描述列表
🔤 内联语义
<a>定义超链接内联语义
<abbr>定义缩写或简称内联语义
<b>定义粗体文本（无语义重要性）内联语义
<bdi>定义文本的隔离，用于处理不同方向（如从右到左）的文本内联语义
<bdo>覆盖文本的方向内联语义
<br>定义换行内联语义
<cite>定义作品标题（如书、文章、歌曲的标题）内联语义
<code>定义计算机代码文本内联语义
<data>将内容与机器可读的翻译相关联内联语义
<dfn>定义术语的定义实例内联语义
<em>定义强调文本（通常斜体显示）内联语义
<i>定义斜体文本（用于技术术语、外语、思想等）内联语义
<ins>定义插入的文本内联语义
<del>定义删除的文本内联语义
<kbd>定义键盘输入内联语义
<mark>定义标记或高亮的文本内联语义
<q>定义短的引用（行内引用，浏览器自动加引号）内联语义
<rp>定义 ruby 注释在不支持时的后备显示内容内联语义
<rt>定义 ruby 注释的发音或解释内联语义
<ruby>定义 ruby 注释（如中文注音）内联语义
<s>定义不再正确的文本（删除线）内联语义
<samp>定义计算机程序输出的样例内联语义
<small>定义小号文本（如免责声明、版权声明）内联语义
<span>定义内联容器，用于文本分组或样式应用内联语义
<strong>定义重要的文本（通常粗体显示）内联语义
<sub>定义下标文本内联语义
<sup>定义上标文本内联语义
<time>定义时间或日期（可含机器可读格式）内联语义
<u>定义未明确表述但需要标注的文本（如下划线）内联语义
<var>定义变量（数学表达式或程序变量）内联语义
<wbr>定义可能的换行点内联语义
🖼️ 嵌入内容
<area>定义图像映射中的可点击区域嵌入内容
<audio>定义音频内容嵌入内容
<canvas>定义通过脚本（如 JavaScript）绘制图形的区域嵌入内容
<embed>定义外部应用的嵌入容器（如插件、Flash）嵌入内容
<iframe>定义内联框架，用于嵌入另一个 HTML 页面嵌入内容
<img>定义图像嵌入内容
<map>定义图像映射（与 <area> 配合）嵌入内容
<object>定义外部资源（如图像、插件、PDF）嵌入内容
<param>定义 <object> 元素的参数嵌入内容
<picture>定义多个图像源的容器，用于响应式图片嵌入内容
<source>定义 <picture>、<audio> 或 <video> 的媒体资源嵌入内容
<svg>定义可缩放矢量图形嵌入内容
<track>定义 <video> 或 <audio> 的字幕轨道嵌入内容
<video>定义视频内容嵌入内容
📊 表格
<table>定义表格表格
<caption>定义表格的标题表格
<colgroup>定义表格列组表格
<col>定义表格列的属性表格
<thead>定义表格的表头部分表格
<tbody>定义表格的主体部分表格
<tfoot>定义表格的表尾部分表格
<tr>定义表格行表格
<th>定义表格的表头单元格表格
<td>定义表格的数据单元格表格
详细
📝 表单
<form>定义用户输入的表单表单
<input>定义输入控件（类型包括 text、password、checkbox 等）表单
<textarea>定义多行文本输入控件表单
<button>定义可点击的按钮表单
<select>定义下拉列表表单
<optgroup>定义 <select> 中的选项组表单
<option>定义 <select> 中的选项表单
<label>定义 <input> 元素的标签表单
<fieldset>定义表单中相关的控件组表单
<legend>定义 <fieldset> 元素的标题表单
<datalist>定义输入控件的预定义选项表单
<output>定义计算结果表单
<progress>定义任务的完成进度表单
<meter>定义度量衡（如磁盘使用量、评分）表单
🖱️ 交互元素
<details>定义用户可查看或隐藏的额外细节交互元素
<summary>定义 <details> 元素的可见标题交互元素
<dialog>定义对话框或窗口交互元素
⚙️ 脚本和模板
<script>定义客户端脚本（如 JavaScript）脚本和模板
<noscript>定义在不支持脚本时的替代内容脚本和模板
<template>定义通过脚本实例化的 HTML 模板脚本和模板
<slot>定义 Web Component 的插槽脚本和模板

🔍 每个类别前都有深色背景的分组标题，标签中的 < 和 > 已转义为实体，可直接复制使用。
